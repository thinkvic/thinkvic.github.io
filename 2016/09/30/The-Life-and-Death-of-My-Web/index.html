The Life and Death of My Web

Sep 30, 2016

My website was created in 2014.
It went though many rounds of rebirth.
Initially, it’s some static web pages with an integrated WordPress blog. The frontend is mostly JQuery-based.

In 2015, I restructured the codes to make it simpler and dynamic. PHP is used for backend templating.
In 2016, I revamped both my website and blog.
Now the website has an AngularJS frontend and a NodeJS backend.
It also contains an external link to the blog.
The blog is now built on top of Hexo, a simple framework powered by NodeJS.
